# Part 1: Neural Network Analysis

## Overview
This project builds and analyzes a feed-forward neural network to predict customer churn.

## Dataset
Source: [Masai Module 5 Dataset](https://drive.google.com/drive/folders/1akV6po4Nrgkc3yQrJkzA6cJlV-wBvUYs?usp=sharing)
File: `customer_churn_nn.csv` — 2000 rows, 17 columns

## Tasks Completed
- Task 1: Dataset exploration and class distribution analysis
- Task 2: Preprocessing — encoding, scaling, train-test split
- Task 3: Feed-forward neural network (64→32→1)
- Task 4: Training with class weights, confusion matrix, classification report
- Task 5: 4 hyperparameter experiments with comparison table
- Task 6: Final reflection on weights, activation, learning rate, overfitting

## Results
- Best model: Deeper Network [128,64,32] — Test Accuracy: 97.5%
- Results saved in `results/` folder
